# Start Session

用于开始一次非琐碎开发会话。

## 步骤

1. 读取 `CLAUDE.md`、相关 `.claude/` 规则和用户需求。
2. 如果存在，读取 `tasks/progress.md`、`tasks/feature-list.json`、`tasks/session-handoff.md`。
3. 确认当前最高优先级未完成功能、已有 `in_progress` 项和 blocker。
4. 只选择一个明确功能作为本轮目标。
5. 建立冲刺合同：目标、范围、排除项、验收标准、验证方式和预计改动范围。
6. 确认需要的运行时信号和过程证据，例如测试输出、日志、健康检查或截图。
7. 如果标准启动或标准验证当前失败，先处理基础状态。

## 输出

- 本轮目标
- 当前状态
- 计划步骤
- 排除项
- 验证方式
- 需要记录的证据
- 已知风险
